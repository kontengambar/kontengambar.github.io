Tutorial Instalasi Aplikasi Intra

Langkah 1

Buka aplikasi Play Store di perangkat Android Anda.

Langkah 2

Cari "Intra" menggunakan kolom pencarian di Play Store.

Langkah 3

Ketika Anda menemukan aplikasi Intra, ketuk ikon aplikasi tersebut untuk membuka halaman detail.

Langkah 4

Di halaman detail aplikasi, ketuk tombol "Instal" untuk memulai proses instalasi.

Langkah 5

Tunggu hingga proses instalasi selesai. Setelah selesai, Anda dapat membuka aplikasi Intra dari layar beranda atau menu aplikasi.
